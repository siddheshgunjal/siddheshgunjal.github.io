FluxTest

Description

  • Siddhesh Gunjal

  •   Github link

FluxTest is a self-hosted server network test application that measures:

  • • Latency (round-trip time)
  • • Download throughput
  • • Upload throughput

The app serves a browser UI and exposes HTTP endpoints for health checks and throughput testing between a client and your own hosted server. It is designed to run in Docker.
